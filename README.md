Super Store Sales Dashboard :
Project Overview :
This project is an interactive **Power BI dashboard** developed to analyze Super Store sales data. It provides valuable insights into sales performance, profit trends, customer behavior, regional performance, and shipping analysis through dynamic visualizations.

 Key Features :
- Sales and Profit Analysis
- Regional Performance Tracking
- Customer Segment Analysis
- Category and Sub-Category Insights
- Shipping Mode Analysis
- Interactive Filters and Slicers
- KPI Cards and Dynamic Charts

 Tools & Technologies Used :
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
